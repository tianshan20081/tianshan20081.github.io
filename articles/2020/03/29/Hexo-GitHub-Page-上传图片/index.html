    使用相对路径

    需要 配置 /_config.yml
    post_asset_folder: true
    然后 执行命令 hexo n post_name
    会在 source/_posts 下面 出现一个 post_name 文件夹和 post_name.md 的文件
    将图片存放在 post_name 文件夹 下面即可
